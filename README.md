# AorticHeartValveBlenderAddon

## Create_aortic_valve
Скрипт создает анатомически точную развертку протеза для аортального клапана сердца.
- Выбор диаметра аорта
- Выбор толщины
- Возможность экспорта в STL с корректным масштабом

## length
Скрипт считает сумму выделенных ребер. Scale объектов должен был приведен к 1.
